# Privantix Website (Bootstrap)
Sitio multipágina estático (HTML + Bootstrap CDN) para Privantix.

## Estructura
- index.html: landing principal
- product.html, roadmap.html, vision.html, open.html, contact.html
- legal/privacy.html, legal/terms.html
- assets/css/style.css
- assets/js/main.js
- assets/img/logo.svg

## Cómo abrir
Abre `index.html` en tu navegador. Si quieres navegación sin restricciones de CORS (no es necesario aquí), puedes servirlo con un servidor local:
- Python: `python -m http.server 8000` y visitar http://localhost:8000
